# Math Trainer Demo

Демоверсия интерактивного математического тренажёра для 5 класса.

## О проекте

Тренажёр помогает школьникам отработать основные типы задач по математике в удобном игровом формате.  
В демоверсии доступно по 2 задания в каждом разделе.

## Разделы демоверсии

- Движение в разных направлениях
- Приведение к единице
- Площадь, периметр и объём
- Задачи через уравнение
- Уравнения с десятичными дробями
- Действия с десятичными дробями
- Логические задачи

## Возможности

- проверка ответа;
- подсказки;
- объяснение решения;
- счётчик правильных и неправильных ответов;
- светлая и тёмная тема;
- игровой персонаж Магистр Числус.

## Назначение

Демоверсия создана для знакомства с форматом тренажёра.  
Полная версия содержит расширенный банк заданий и уровни сложности.
